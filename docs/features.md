# Возможности платформы ByteKids

Краткий обзор для партнёров и школ. Полный функционал доступен в продакшене; в демо — вымышленные данные и ограничения на запись.

## Роли

### Администратор

- Список учеников и групп, архив удалённых
- Монеты, оплаченные занятия, история баланса
- Записи с форм сайта и мастер-классов
- Договоры на обучение (подписание, PDF)
- Заказы из магазина, ожидающие оплаты
- Аналитика и дашборд
- Настройки офиса, демо-доступы (в продакшене)
- Печать карточек учеников с QR-кодом

### Преподаватель

- Ученики только своих групп
- Расписание (общее + личные события)
- Карточки и QR учеников
- Доход и начисления
- Замечания и поощрения (в рамках прав)

### Ученик

- Вход по логину / QR
- Модули и задания
- Личный прогресс

## Демо-режим (`/demo`)

Предназначен для ознакомления без доступа к реальным данным школы.

| Параметр | Значение |
|----------|----------|
| Срок доступа | 12 часов с момента регистрации |
| Роли | admin, teacher, student (3 аккаунта) |
| Данные | 5 вымышленных учеников, 2 группы, расписание, уведомления |
| Изоляция | Отдельный демо-офис, без влияния на продакшен |
| Ограничения | Нет сохранения критичных настроек, часть API только чтение |

После регистрации на [bytekids.online/demo](https://bytekids.online/demo) выдаются логины для всех трёх ролей.

## Интеграции (продакшен)

- Telegram-бот для родителей
- VK (родительская панель, уведомления)
- Приём оплат
- QR-отметка посещений

В демо интеграции с внешними сервисами не затрагивают реальных пользователей.

## Мультиофис

Платформа поддерживает несколько филиалов (офисов) с раздельными данными и расписанием. Подходит для сетей школ.
